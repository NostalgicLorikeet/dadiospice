random_use_old_version

Use the deprecated random number generator or the regular one.

Syntax:

random_use_old_version(flag);


ArgumentDescription
flag Use the deprecated seed generator (true) or the regular one (false).


Returns: Real


Description

The GameMaker: Studio random number generation was improved and the previous generator deprecated from use. However if you have a game that requires seeds generated from the old generator, then you will need to use this function to switch the regular generator off and use the deprecated version instead.


Example:

random_use_old_version(true);

The above code will set the random number generator to use the deprecated version.
