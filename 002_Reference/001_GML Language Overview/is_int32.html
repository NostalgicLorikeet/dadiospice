is_int32

Returns whether the given argument is an int32.

Syntax:

is_int32(n);


ArgumentDescription
n The argument to check.


Returns: Boolean


Description

This function returns whether a given variable is a 32bit integer or not. In some cases you want to check and see what data type a variable holds in GameMaker: Studio and that's when you would use this function. It returns true or false depending on whether the value is an int 32 or not. For further information on data types see here.


Example:

if !is_int32(val)
   {
   show_debug_message("Not a 32 bit integer!");
   }

The above code checks the variable "val" to see if it contains an int32 and if it is not then it shows a message in the debug console.
